# Prefabs

The starter scene creates primitive placeholders at runtime so it can import without binary assets. It uses a sphere for the fire orb projectile and a capsule for the homing pulse projectile so the two projectile paths are distinct before real assets exist. Use this folder for project-specific core, enemy, weapon, projectile, hitscan tracer, impact VFX, and audio-ready prefabs after replacing the generated placeholders.

Attack presentation events tolerate missing optional audio and VFX references, so the sample remains playable before production assets exist.
